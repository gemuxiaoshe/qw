基于github基础上的个人博客搭建心得。

1.需要用到的基本工具和技能

  • GIThub网站
  • Github Desktop
  • 一个记事本
  • 基本的HTML知识

    • GitHub是一个强大的代码托管平台，它为众多的开发者提供了一个代码共享和交流的平台，你可以在这发现很多有意思的代码，和别人做的项目，你可以借鉴，copy和学习。并且它会给每个用户免费提供一个500兆的储存空间，并分配一个特定的域名，来显示你的网页，这也是能够使用github搭建一个纯免费的个人网页，或者个人博客的根本原因。它的优点是：

    • 完全免费的，不需要任何费用。
    • 不用购买虚拟主机，也可以不用购买域名。你就可以搭建一个可以被访问的个人网页和博客。
    • 十分自由的，只要你具备相关的知识，你就可以做出任何你想要的形式的网站，没有局限性。
    • 对编程网页设计是十分有用的，你可以直接编辑代码，而不是拖动标签，添加按钮来设计你的页面，你也可以在本地写好一个网页直接上传。以此来分享你的成果。

    GitHub Desktop是github网站的一个本地软件程序，它主要的作用是帮助我们实现代码的本地和github网站之间的一个远程管理.
